About This System

Environmental Sensor Monitoring System

This web application provides authenticated access to Industrial Control System (ICS) environmental sensor data. It is designed to serve personnel who require visibility into facility environmental conditions, displaying temperature and relative humidity readings collected over the previous 24 hours.

Key Features:
  • Real-time Monitoring: View current temperature and humidity readings from ICS environmental sensors
  • Historical Data: Access 24-hour historical data with summary statistics
  • Cloud Backup: Automatic synchronization with cloud archival services
  • Data Import: Import sensor readings from external data files
  • Role-based Access: Separate user and administrator privileges
  • Audit Logging: Comprehensive logging of all system access and actions
User Roles:
  • User: Read-only access to view environmental data and system status
  • Administrator: Full access including the ability to insert, update, and delete sensor records
